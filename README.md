# LBCC Agent — Replit

## Como rodar (passo a passo)

### 1. Acesse replit.com e crie conta grátis

### 2. Novo Repl → Import ZIP
- Clique em **"+ Create Repl"**
- Escolha **"Upload folder"** ou **"Import ZIP"**
- Faça upload do `lbcc-agent-replit.zip`

### 3. Adicionar chave OpenAI
No menu lateral clique em **🔒 Secrets**:
```
Key:   OPENAI_API_KEY
Value: sk-sua-chave-aqui
```

### 4. Clicar em ▶ Run
Instala tudo e inicia automaticamente.

### 5. Abrir Webview
Clique na aba **Webview** — a interface do agente abre no browser.

### 6. Testar
Digite: **Abra o Google**

O agente navega (headless) e mostra o screenshot no chat.
